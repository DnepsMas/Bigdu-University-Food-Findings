# BUFF Archive

这里是《大邮食堂志》（*Bigdu[^1] University Food Findings*）的归档仓库。

读者可以在右侧 Releases 栏中找到最新版本。

本仓库包含《大邮食堂志》的源码；实际上，《大邮食堂志》目前开放源码和社区 PR。如果您想参与《大邮食堂志》的建设，请直接参加!

## Issue

若您对本书有任何补充意见，或发现本书中的内容错误，包括但不限于排版问题、语病、错别字或其他 typo、菜品信息（价格、分量、口味描述）的偏差、窗口的关闭或迁移，请参照 [Community](#community) 一节联系我们。

特别提示：食堂菜品的价格与供应情况可能因季节、校历或食堂方政策随时调整，本书所记录的信息仅代表实地调查时的情况，恕无法保证实时准确——这一点与数学题的答案有本质区别。

## Contribution

若您想提供菜品信息、实地拍摄、点评撰写、排版协助或宣传推广，抑或是想帮我们跑一趟尚未收录的食堂楼层（可借助 AI 润色点评，但您需对真实性负责），请参照 [Community](#community) 一节联系我们。衷心感谢您的**无偿**贡献，您将有权把自己加入贡献者列表中。

## Roadmap

本文档将持续收集反馈与菜品投稿，并发布新版本；也乐于接受更多食堂楼层、特色窗口乃至校内小卖部的资料。

请大家理解编者现实的精力限制：单人完成全校食堂的调研与写作已属不易，因此我们暂时无法保证每道菜品都配有实拍图，也无法频繁核实价格变动，还望见谅。

本文档目前以学生食堂为主要收录范围，暂不纳入教工餐厅，但不排除日后另立专题。

若没有更多朋友协助，我们可能在不远的将来（也许还没毕业？）启动《大邮周边探店志》等衍生项目，敬请期待。

本书有计划在日后将排版模板的源码独立公开，但暂未计划开放内容的源码。

## Footnotes

[^1]: 「Bigdu」的命名沿袭自《大邮数学集》（BUMP）的传统——后者解释道，此称呼不过是为了凑出一个 B 字头。我们在此基础上更进一步：B(igdu) U(niversity) F(ood) F(indings) 恰好缩写为 BUFF，既是「充实强健」之意，也是吃好食堂后理想的精神面貌，一举两得。
